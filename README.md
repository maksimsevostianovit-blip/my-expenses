# My-Expenses

## Описание

Тестовое задание. Контроль финансов. Реализована вкладка "Мои расходы" <br />
Адаптивная вёрстки нет. На разрешение меньше **1295px** будет отображаться заглушка <br />
[Живая страница](https://maksimsevostianovit-blip.github.io/my-expenses).

## Зависимости

На компьютере должена быть установлена LTS-версия [Node.js](https://nodejs.org/en/).

## Перед началом работы

Один раз нужно установить все зависимости.

```shell
npm i
```

### Разработка

Запустить режим разработки.

```shell
npm run start
```

### Сборка

Запустить режим разработки.

```shell
npm run build
```
